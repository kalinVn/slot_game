# Slot_game

1. Install
    npm install

2. Start Game
    npm run start

3. Build Game
    npm run game

4. Entry point 
    src/index.js

5. When have line in game over window animation is showed as line with three animated symbols below message. 
   If win 2 or 3 lines in game over is show 2 or 3 lines with current animated symbols.
